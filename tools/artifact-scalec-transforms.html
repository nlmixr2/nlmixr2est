FOCEi · parameter scaling diagnostics

Guarding each transform's scaleC — per parameter, per bound

FOCEi rescales every population parameter by a constant scaleC so the optimizer sees order-unity variables. Each transform has its own derivative-based formula, and FOCEi guards it: inside a healthy band the formula is kept (results preserved); in the transform's singular region — where scaleC collapses to 0, diverges, or flips sign — it falls back to the native magnitude |init|, and for a bounded transform, if |init| is also out of range, to the band's geometric middle.

For a bounded transform the band is not a fixed cutoff — it is built from this parameter's own low and high bound. A derivative-based scaleC factors as N · M: a per-parameter natural scale N that uses the distance to each bound, times a bounds-invariant factor M that carries the singularity. Guarding scaleC to N · [lo, hi] applies the same dimensionless band at every bound — so logit(x,0,1) and logit(x,1,100) are guarded identically at equal fractional position, and a wide interval keeps its healthy (large) scaleC instead of being clipped by a one-size band.

Bounded transforms — the band tracks the bounds. Each pair below is the same transform at a narrow and a wide/offset interval. The green band is a curve now (N(x)·[lo,hi]), not a flat line: it lifts with the interval so the healthy scaleC stays inside it at every bound. A fixed band would clip the wide interval's legitimate large values and slam them to the midpoint.

Unbounded / structural transforms — fixed bands. These parameters have no interval; M is already dimensionless, so the band is a constant. Shown for reference (unchanged).
