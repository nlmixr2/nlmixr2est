# Extract residuals from the FOCEI fit

Extract residuals from the FOCEI fit

## Usage

``` r
# S3 method for class 'nlmixr2FitData'
residuals(
  object,
  ...,
  type = c("ires", "res", "iwres", "wres", "cwres", "cpred", "cres")
)
```

## Arguments

- object:

  focei.fit object

- ...:

  Additional arguments

- type:

  Residuals type fitted.

## Value

residuals
