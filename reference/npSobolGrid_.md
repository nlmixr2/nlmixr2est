# Sobol initial grid over a box (nonparametric engines)

Sobol initial grid over a box (nonparametric engines)

## Usage

``` r
npSobolGrid_(n, lower, upper)
```

## Arguments

- n:

  Number of support points.

- lower, upper:

  Numeric vectors giving the per-dimension box bounds.

## Value

Numeric matrix, one support point per row.
