# Messages the nlmixr2 logo...

Messages the nlmixr2 logo...

## Usage

``` r
nlmixr2Logo(str = "", version = sessionInfo()$otherPkgs$nlmixr2$Version)
```

## Arguments

- str:

  String to print

- version:

  Version information (by default use package version)

## Value

nothing; Called to display version information
