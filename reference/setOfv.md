# Set/get Objective function type for a nlmixr2 object

Set/get Objective function type for a nlmixr2 object

## Usage

``` r
setOfv(x, type)

getOfvType(x)
```

## Arguments

- x:

  nlmixr2 fit object

- type:

  Type of objective function to use for AIC, BIC, and \$objective

## Value

Nothing
