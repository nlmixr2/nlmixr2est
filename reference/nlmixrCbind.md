# nlmixrCbind

\`cbind\` for \`nlmixr\` objects that preserve the fit information

## Usage

``` r
nlmixrCbind(fit, extra)
```

## Arguments

- fit:

  nlmixr fit

- extra:

  data to cbind to nlmixr fit

## Value

fit expanded with extra values, without disturbing the fit information
