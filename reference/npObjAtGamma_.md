# Diagnostic: NPAG objective at a fixed grid and residual multiplier gamma

Diagnostic: NPAG objective at a fixed grid and residual multiplier gamma

## Usage

``` r
npObjAtGamma_(etaPoints, cores, gamma)
```

## Arguments

- etaPoints:

  support points, one per row

- cores:

  threads

- gamma:

  residual-error multiplier

## Value

offset-corrected marginal log-likelihood
