# This is a S3 method for getting the distribution lines for a base rxode2 nls problem

This is a S3 method for getting the distribution lines for a base rxode2
nls problem

## Usage

``` r
rxGetDistributionNlsLines(line)

# S3 method for class 'norm'
rxGetDistributionNlsLines(line)

# Default S3 method
rxGetDistributionNlsLines(line)
```

## Arguments

- line:

  Parsed rxode2 model environment

## Value

Lines for the focei. This is based on the idea that the focei parameters
are defined
