# Drop mu referenced etas and covariates

Drop mu referenced etas and covariates

## Usage

``` r
.saemDropMuRefFromModel(ui, noCovs = FALSE)
```

## Arguments

- ui:

  rxode2 ui

- noCovs:

  Do not look for covariates

## Value

model line expression with mu referenced information dropped.
