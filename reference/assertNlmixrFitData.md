# Assert that this is a nlmixr2 fit data object

Will error without nlmixr2 fit data object

## Usage

``` r
assertNlmixrFitData(fit)
```

## Arguments

- fit:

  Fit object

## Value

Nothing

Matthew L. Fidler

## Examples

``` r
if (FALSE) { # \dontrun{

f <- 4
assertNlmixrFitData(f) # throw errors

} # }
```
