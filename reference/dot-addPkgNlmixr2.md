# Adds a package to the nlmixr2's \$sessioninfo inside the fit

Adds a package to the nlmixr2's \$sessioninfo inside the fit

## Usage

``` r
.addPkgNlmixr2(pkg)
```

## Arguments

- pkg:

  character vector of the package to add

## Value

nothing, called for side effects

## Author

Matthew L. Fidler

## Examples

``` r
.addPkgNlmixr2("nlmixr2") # already present
```
