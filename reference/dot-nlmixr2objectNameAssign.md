# Allows external methods (like those in nlmixr2) to assign object name

Allows external methods (like those in nlmixr2) to assign object name

## Usage

``` r
.nlmixr2objectNameAssign(x)
```

## Arguments

- x:

  String or null for assigning a nlmixr object name

## Value

nothing called for side effects
