# Adjust covariance matrix based on scaling parameters

Adjust covariance matrix based on scaling parameters

## Usage

``` r
.nlmAdjustCov(cov, parScaled)
```

## Arguments

- cov:

  Covariance of scaled parameters

- parScaled:

  The final scaled parameter value

## Value

The adjusted covariance matrix based on the scaling
