# List all pre-processing hooks

List all pre-processing hooks

## Usage

``` r
preProcessHooks(name = NULL)
```

## Arguments

- name:

  when specified, the name of the hook, otherwise \`NULL\` to list all
  hooks

## Value

a charcter vector listing all pre-processing hooks or the function for
the hook
