# Assert that this is a nlmixr2 fit object

Will error without nlmixr2 fit object

## Usage

``` r
assertNlmixrFit(fit)
```

## Arguments

- fit:

  Fit object

## Value

Nothing

## Author

Matthew L. Fidler

## Examples

``` r
if (FALSE) { # \dontrun{

f <- 4
assertNlmixrFit(f) # throw error

} # }
```
